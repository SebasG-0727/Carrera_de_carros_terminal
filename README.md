Es una carrera de carros realizada en el lenguaje de Python para visualizar en terminal.

El juego contiene tres pistas diferentes, donde el jugador elige un carro, estos carros se mueven automaticamente acelerados por un conductor que aumenta el recorrido del carro de forma aleatoria por medio de un dado. Si el carro que eligio el jugador gana entonces este jugador habra ganado. la lista de los carros ganadores se guardan en un archivo ".txt".
